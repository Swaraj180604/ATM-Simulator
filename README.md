# ATM Simulator - Python Application
A console-based ATM simulation system built in Python that allows users to create accounts, perform secure logins, and execute banking transactions with data persistence using Excel files.

## Features
**Account Management**: Create new bank accounts with name, PIN, and initial balance

**Secure Login**: Authenticate users using account number and PIN

**Banking Operations**:

💰 **Deposit**: Add funds to your account

💸 **Withdraw**: Remove funds with insufficient balance protection

📊 **Balance Check**: View current account balance

**Transaction History**: Automatically records all transactions with timestamps

**Data Persistence**: All account and transaction data stored in Excel files

## Technical Highlights
Built with Python

Uses openpyxl for Excel file operations

Implements pandas for data visualization

Transaction logging with datetime stamps

Menu-driven console interface

# Technologies Used
Python 3

openpyxl

pandas

datetime

## Files Generated
accounts.xlsx - Stores user account information

transactions.xlsx - Maintains transaction history

Perfect for learning Python basics, file handling, and building simple banking system logic!
